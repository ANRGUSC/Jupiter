
Welcome to Jupiter!
===================================

`Jupiter`_ is a Central orchestrator for Dispersed Computing that uses `Docker`_ containers and `Kubernetes`_ (K8s). 

The code is open source, and `available on GitHub`_.

.. _Jupiter: http://jupiter.readthedocs.io/en/latest/
.. _Docker: https://docs.docker.com/
.. _Kubernetes: https://kubernetes.io/docs/home/
.. _available on GitHub: https://github.com/ANRGUSC/Jupiter
